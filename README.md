PT-BR

Aplicação Windows Forms com CRUD completo de usuários, paginação, filtros, validações e arquitetura organizada em camadas (MVC simplificado). Com 2 tipos de conexão à banco de dados, MySQL e SQLite(local). Desenvolvido para estudos de C#, .NET e boas práticas de desenvolvimento desktop.


EN

Windows Forms application providing a complete user management system (CRUD) with pagination, filtering, validations and a clean layered architecture. It has 2 types of database connection, MySQL and SQLite(local). Built for studying C#, .NET and desktop development best practices.
